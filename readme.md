## Clustering images using scikit learn

The repository holds clustering performed on images in python.

### Prerequisites

+ Python 3
+ Jupyter notebook
+ Scikit learn

### Algorithms in notebook

+ KMeans Clustering
+ Density based clustering (DBScan)
